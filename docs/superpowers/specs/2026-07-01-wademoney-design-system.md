# WadeMoney — 디자인 시스템 & 화면 사양

- **작성일**: 2026-07-01
- **출처**: Claude Design 핸드오프 번들 (`docs/design/app-design-specification-analysis/`)
- **역할**: 구현이 **픽셀 단위로 따라갈** 시각 레퍼런스. 기능/로직은 [`2026-07-01-simple-budget-ios-design.md`](./2026-07-01-simple-budget-ios-design.md) 참조.
- **원본 프로토타입**: HTML/CSS/JS (렌더 아님, 소스에서 값 추출). 대표 파일 `project/WadeMoney 가계부.dc.html`, 공유 데이터 `project/wade-data.js`.

> 프로토타입 매체는 HTML이다. **시각 결과물을 SwiftUI로 재현**하되 내부 구조는 SwiftUI에 맞게 짠다.

## 0. 브랜드

- **앱 이름**: WadeMoney
- **태그라인**: "가볍게 기록하는 하루 지출 · 온디바이스 가계부"
- **성격**: 따뜻한 크림/베이지 배경 + 제이드 그린 포인트. 둥근 모서리, 부드러운 그림자, 친근함.
- **한글 폰트**: **Pretendard Variable** (무료, 앱에 번들). 숫자는 `tabular-nums`(고정폭) 적용.
- **아이콘 폰트**: 프로토타입은 **Material Symbols Rounded**(FILL 변형) 사용. → §9 아이콘 세트 결정 참조.

## 3. 형태 토큰 (radius / shadow / spacing)

- **Radius**: 큰 카드 `24` · 리스트/그룹 카드 `20` · 키/카테고리 카드·입력칸 `16` · 세그먼트/메모 `14` · 아이콘 타일 `10–12` · 알약 `999`. 바텀시트 상단 `30`. FAB `20`.
- **Shadow**: 카드 `0 1px 2px rgba(0,0,0,.04), 0 10px 26px var(--shadow)` · 리스트 `0 8px 22px var(--shadow)` · FAB `0 8px 20px var(--primaryglow)` · 내부 보더 `inset 0 0 0 1px var(--line)`.
- **Spacing**: 화면 좌우 패딩 `18px`. 스크롤 영역 상단 `60`, 하단 `104`(탭바 공간). 카드 간 `14`. 카드 내부 패딩 `18–20`.
- **탭바**: `backdrop-filter: blur(18px) saturate(180%)`, 상단 `0.5px` line, 패딩 `10px 26px 26px`.

## 5. 화면 사양

### 5.1 대시보드 — "한눈에"
1. 헤더: 대제목 "한눈에" + 우측 **리포트** 버튼(`auto_awesome`, primarysoft 알약) → AI 리포트로 이동.
2. **기간 세그먼트** `일 / 월 / 연` (슬라이딩 알약, `transform` 애니메이션).
3. **기간 이동**: ◀ `{기간 라벨}` ▶ (원형 버튼). 라벨 예: "2026년 7월", "7월 15일 (오늘)", "2026년".
4. **히어로 예산 카드**:
   - 좌: **소진율 링**(`conic-gradient(primary … , track …)`), 중앙에 `{pct}%` + "소진".
   - 우: scope 라벨("이번 달 총지출") · `₩{총액}`(30/800) · **페이스 배지**.
   - **페이스 배지**(월/연): `{±%}` + 화살표. 더 씀→`bad`/`badsoft`/`arrow_drop_up`, 덜 씀→`good`/`goodsoft`/`arrow_drop_down`. 아래 note "지난달 같은 시점보다"(연: "작년 같은 시점보다").
   - **일 뷰**: 페이스 대신 "일예산 여유" 배지 + "일예산 {X}원 중 {Y}원 남음".
   - 하단: 프로그레스 바 + "예산 {X}원 · {Y}원 남음".
5. **AI 인사이트 카드**(aiOn일 때): aitint 그라데이션, `auto_awesome` + "AI 인사이트" + 우측 태그 알약(양호=good/주의=bad). 본문 1–2문장. "자세히 보기 ›" → 리포트.
6. **카테고리 비중 카드**: `conic-gradient` 도넛(중앙 홀 `inset 19px`, 중앙에 "총지출"+금액) + 범례(색점·이름·%). 범례/슬라이스 탭 → 내역 해당 카테고리 필터로 이동.
7. **지출 추세 카드**: 막대 그래프. 현재 기간=`primary`+값 라벨 강조, 과거=`barmuted`, 미래(0)=`track`. note: 월="최근 6개월", 연="2026년 월별", 일="최근 7일".

### 5.2 내역
- 대제목 "내역" + **가로 스크롤 필터 칩**(전체 · 각 카테고리 · 수입). 선택 칩 `primary`/흰 글자.
- **날짜 그룹**: 헤더(날짜 + `오늘/어제` 태그 알약 + 그룹 합계). 지출 합 `−`, 순수입일 `+`(good).
- **행**: 아이콘 타일(`color+'22'` 배경, 카테고리 색 아이콘) · 이름 · "{카테고리} · {시간}" · 금액(지출 `−` ink / 수입 `+` good). 수입 아이콘 `trending_up`, goodsoft 타일.
- **빈 상태**: `receipt_long` 원형 + "아직 기록이 없어요" / "+ 버튼으로 첫 지출을 기록해보세요"(필터 시 문구 변형).

### 5.3 설정
- 섹션 라벨(ink3, 700) + 카드 그룹.
- **예산**: "이번 달 예산" 행(우측 `₩{금액}` + ›, 탭→예산 시트) · "월 시작일 매월 1일".
- **카테고리 · AI**: "카테고리 관리 {n개} ›" · "AI 기능 (온디바이스 · Apple Intelligence)" + **토글 스위치**(51×31, 켜짐 primary).
- **동기화 · 데이터**: "iCloud 동기화 · 방금 전 동기화됨"(good, `cloud_done`) · "CSV 내보내기 ›".
- 푸터: "WadeMoney v1.0 · 데이터는 이 기기에만 있어요".

### 5.4 카테고리 관리 (설정 하위)
- 뒤로(‹ 설정) + 제목 "카테고리 관리".
- **사용 중**: 행마다 `drag_indicator`(재정렬) · 아이콘 타일 · 이름 · "이번 달 {금액}원"/"사용 없음" · ›.
- **보관됨**(아카이브): opacity 0.62, "과거 내역엔 유지돼요" + **복원** 버튼.
- 하단: 점선 테두리 "**+ 새 카테고리**".

### 5.5 AI 리포트 (대시보드 하위)
- 뒤로(‹ 대시보드) + `auto_awesome` + "{월} 소비 리포트" + "2026년 7월 · 15일 경과".
- **요약 카드**(aitint 그라데이션): 태그(주의/양호) + 서술문. 핵심 수치는 색 강조(예: "32% 늘었고").
- **예상 지출 카드**: "이번 달 예상 지출 ₩{예상}" + "예산 초과 예상 +{초과}원" 배지(bad).
- **지난달 대비 변화**: 카테고리별 행(아이콘 · 이름 · `{±%}` 화살표, 증가 bad / 감소 good).
- **팁 카드**(primarysoft): `lightbulb` + 실행 제안("카페를 2번만 줄여도 …").
- 푸터: `lock` "온디바이스에서 생성됨 · 데이터는 기기를 벗어나지 않아요".

### 5.6 빠른 입력 시트 (바텀시트)
- `sheetUp` 애니메이션, 상단 그랩 핸들(38×5), 딤 배경.
- 헤더: 제목("새 지출"/"새 수입"/"이번 달 예산") + **지출/수입 세그먼트**(지출=primary, 수입=good 알약).
- **금액 디스플레이**: `₩` + `{금액}`(52/800), 0일 때 ink3.
- **진행 단계 칩**(입력 모드): 금액 → 카테고리 → 저장. 현재 도달 단계 primarysoft/primary.
- **카테고리 그리드**(지출 & 입력 모드): 4열, 카드마다 아이콘 타일 + 이름. 선택 시 primarysoft + `inset 0 0 0 2px primary`.
- **메모 입력** + **AI 다듬기** 버튼(`auto_awesome`). 다듬은 후: "메모를 다듬고 **{카테고리}** 카테고리를 추천했어요"(aitint2 안내). 버튼 라벨 "AI 다듬기"→"정리됨".
- **키패드**: 3열, `1–9, 00, 0, backspace`(56 높이, card2). 빈 값에 `0/00` 무시, 최대 10자리.
- **저장 버튼**: 유효 시 primary/흰 글자, 아니면 track/ink3. 라벨 "저장하기"/"예산 저장".
- **유효성**: 입력 모드 = `금액>0 && (수입 || 카테고리 선택)`. 예산 모드 = `금액>0`. 수입은 카테고리 없음.
- 저장 후: 시트 닫힘 + **토스트**("저장했어요"/"예산을 …원으로 저장했어요"), 햅틱.

### 5.7 공통
- **토스트**: 화면 하단 중앙, toastbg, `check_circle`(good) + 메시지, `toastIn` 1.8s.

## 7. 위젯 & 앱 밖 진입점

- **홈 요약 위젯**: "오늘 지출 ₩{X}" + 프로그레스 + "이달 예산 {Y}원 남음".
- **홈 대화형 기록 위젯**: 카테고리 칩(카페·식비·교통·**직접**) — 탭하면 **앱을 열지 않고 금액만 입력해 기록**(App Intents 기반 interactive widget).
- **잠금화면 위젯**: 원형(conic 링 + `savings`) + "남은 예산 ₩{X}". 인라인/원형 액세서리.
- **액션 버튼 / Siri / 단축어**: "커피 4천원 기록"처럼 즉시 저장.
- 위젯·App Intent는 앱과 동일 SwiftData 스토어(App Group) 공유.

## 8. 앱 아이콘

- **콘셉트**: ₩ 표시가 박힌 도넛(브랜드 컬러 conic)을 **베어 무는 분홍 돼지**, 크림 그라데이션 배경(`#FBF6EC→#F1E6D5`). 슈퍼타원 라운드(≈22.5%).
- 돼지 핑크 `#EA8FA4 / #EE97AB`, 도넛 conic에 제이드/골드/블루/핑크. 부스러기 디테일.
- 변형 2종: `WadeMoney 아이콘`, `WadeMoney 아이콘 돼지도넛`.
- **구현**: HTML은 목업. **정식 아이콘 자산(1024px + 필요 사이즈)을 별도 제작해 Assets에 등록**한다(HTML을 코드로 재현하지 않음).

## 9. 구현 시 결정할 것 (열린 항목)

1. **아이콘 세트**: **결정 = Material Symbols Rounded 폰트 번들**(디자인 픽셀 일치).
   - Material Symbols Rounded 변수 폰트(`FILL 0..1`, `wght`)를 앱에 번들(OFL).
   - SwiftUI에선 폰트 리거처/코드포인트로 렌더(예: 작은 `Icon(name:)` 헬퍼로 아이콘 이름→글리프 매핑). 채움은 `FILL 1` 축 사용.
   - 카테고리·UI 아이콘 이름은 §5·§6에 적힌 Material 이름 그대로 사용. SF Symbols 매핑 불필요.
